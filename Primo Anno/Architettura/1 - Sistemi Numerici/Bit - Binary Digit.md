# Cosa è?
Il termine bit rappresenta l'unità di misura fondamentale dell'informazione

Un bit può assumere <u>esclusivamente 2 valori 0 o 1</u>


---
# Unità di misura

| Nome     |             Quantità             |
| :------- | :------------------------------: |
| Byte - B |              8 bit               |
| Word     |              32 bit              |
| Kilo     |     1024 bit = $2^{10}$ bit      |
| Mega     |   1.048.576 bit = $2^{20}$ bit   |
| Giga     | 1.073.741.824 bit = $2^{30}$ bit |
